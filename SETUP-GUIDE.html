⛳ Rosapenna Golf App

Setup guide — estimated time: 15 minutes

You need two free services: Firebase (live database) and GitHub Pages (hosting). Both are free forever for this use case.

Part 1 — Firebase (the live database) ~8 mins

1 Go to console.firebase.google.com and sign in with a Google account.

2 Click Add project. Name it rosapenna-golf-2025. You can turn off Google Analytics — you don't need it. Click through and wait for the project to be created.

3 In the left sidebar, click Build → Realtime Database. Click Create Database. Choose the nearest region (Europe West for Ireland). When it asks for rules, select Start in test mode (you can lock it down after the trip).

4 In the left sidebar, click the gear icon ⚙️ → Project settings. Scroll down to Your apps and click the </> Web icon. Register the app with the nickname rosapenna. You'll see a config block like this:

const firebaseConfig = {
  apiKey: "AIzaSy...",
  authDomain: "rosapenna-golf-2025.firebaseapp.com",
  databaseURL: "https://rosapenna-golf-2025-default-rtdb.europe-west1.firebasedatabase.app",
  projectId: "rosapenna-golf-2025",
  storageBucket: "rosapenna-golf-2025.appspot.com",
  messagingSenderId: "123456789",
  appId: "1:123456789:web:abc123"
};

Copy this entire block — you'll paste it into index.html in a moment.

5 Open index.html in any text editor (Notepad, TextEdit, VS Code, etc.). Find this section near the bottom:

// ── REPLACE THIS BLOCK with your Firebase config ─────────────
const firebaseConfig = {
  apiKey: "YOUR_API_KEY",
  authDomain: "YOUR_PROJECT.firebaseapp.com",
  ...

Replace those 8 lines with your real config block from Firebase. Save the file.

⚠️ The databaseURL line is important — make sure it's in your config. If it's missing, go back to Firebase → Realtime Database and copy the URL from the top of that page (it looks like https://your-project-default-rtdb.region.firebasedatabase.app).

Part 2 — GitHub Pages (free hosting) ~5 mins

6 Go to github.com and create a free account (or log in).

7 Click the + in the top right → New repository. Name it rosapenna-golf. Set it to Public. Click Create repository.

8 On the next screen, click uploading an existing file. Drag and drop ALL the files from your rosapenna-golf folder:

  • index.html (with your Firebase config already in it)
  • manifest.json
  • sw.js
  • The icons/ folder (see note below)

Click Commit changes.

9 Go to Settings → Pages (in the left sidebar). Under Source, select Deploy from a branch, choose main branch, folder / (root). Click Save.

Wait about 60 seconds, then refresh. You'll see a green banner with your URL:

🎉 Your site is live at https://YOUR-USERNAME.github.io/rosapenna-golf/

That's your app URL — share this with all 7 players!

Part 3 — Add the icons (optional but nice)

The app works without icons but the home screen icon will be blank. To add a golf icon, create two square PNG images:

  • icons/icon-192.png — 192×192 pixels
  • icons/icon-512.png — 512×512 pixels

You can use any green golf-themed image. A quick option: go to favicon.io, download the ⛳ emoji as PNG, resize to 192 and 512, and upload to the icons/ folder in your GitHub repository.

Part 4 — Installing to home screen

iPhone (Safari):

  1. Open the app URL in Safari
  2. Tap the Share button (box with arrow)
  3. Scroll down and tap Add to Home Screen
  4. Tap Add

Android (Chrome):

  1. Open the app URL in Chrome
  2. Tap the three dots menu
  3. Tap Add to Home screen
  4. Tap Add

You're done! 🎉

Share the URL with your 6 mates before the trip so they can install it too. All 7 phones will see each other's scores in real time.

💡 Sandy Hills note: The Sandy Hills stroke indexes in the app are estimated. Before Day 2, check the physical scorecard and update the SANDY_HILLS.holes array in index.html if any are different, then re-upload to GitHub.
